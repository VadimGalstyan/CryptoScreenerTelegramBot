CryptoScreenerWithTelegramBot

English:
This is a desktop/console application built with C# that monitors cryptocurrency prices on Binance. Users can set a percentage change and timeframe, and the bot sends alerts via Telegram when the conditions are met. It uses WebSocket for real-time data and parses JSON responses from the API.

Key Features:

Monitor cryptocurrency price changes based on user-defined percentage and timeframe.

Sends notifications directly to Telegram via a bot.

Real-time updates using WebSocket connections.

Handles multiple coins simultaneously.

Simple user interface for setting alerts.

Technologies Used:

C# – Main programming language for the application logic.

WebSocket – For real-time data from Binance API.

JSON parsing (Newtonsoft.Json) – To process API responses.

Telegram Bot API – To send notifications to users.

This project is ideal for crypto traders and enthusiasts who want automated alerts for price changes on Binance.

Russian (Русский):
Это приложение на C#, которое отслеживает изменения цен криптовалют на Binance. Пользователи могут задать процент изменения и таймфрейм, а бот отправляет уведомления в Telegram, когда выполняются условия. Для получения данных в реальном времени используется WebSocket, а JSON-ответы API анализируются программой.

Ключевые особенности:

Мониторинг изменений цен криптовалют по пользовательскому проценту и таймфрейму.

Отправка уведомлений напрямую в Telegram через бота.

Реальное время обновлений через WebSocket.

Поддержка нескольких монет одновременно.

Простой интерфейс для настройки оповещений.

Используемые технологии:

C# – Основной язык программирования для логики приложения.

WebSocket – Для получения данных в реальном времени с Binance API.

JSON-парсинг (Newtonsoft.Json) – Для обработки ответов API.

Telegram Bot API – Для отправки уведомлений пользователям.

Проект идеально подходит трейдерам и энтузиастам криптовалют, которым нужны автоматические уведомления о изменениях цен на Binance.
